# Como utilizar esta API no código final?

Para isto, primeiramente será necessário clonar o repositório e também fazer o download da linguagem [Node.js](https://nodejs.org/pt-br/download). Após feito isso,
no VSCode utilize o seguinte comando no terminal:

`npm install `

Feito isto, irá utilizar os comandos:

``` Power Shell / Terminal
  npx prisma generate
  npx prisma migrate dev

```
Agora, para rodar o servidor é só executar o comando
`node src/server.js`

e teoricamente tudo deve estar concluído e executado corretamente :D
